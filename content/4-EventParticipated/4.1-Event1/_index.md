# Community Day Workshops Report

### Event Objectives

- To share knowledge about Cloud Computing, AI, DevOps, Cyber ​​Security, and Software Development.

- To help students access new technologies and practical experience from speakers.

- To guide career development in the IT field.

### Speaker List

- Viet Phat – GraphRAG with Amazon Bedrock and Amazon Neptune.

- Tran Trung Vinh – From IT Helpdesk to Senior System Administrator.

- Truong Huy Phuoc – The Art of Effective Teamwork.

- Nguyen Quoc Bao – Multiplayer in the Cloud with AWS WebSockets.

- Le Hoang Gia Dai – AWS WAF combined with Machine Learning NIDS.

- Huynh Bao – Docker and Containerization.

### Highlights

#### GraphRAG and Generative AI

- Introduction to Retrieval-Augmented Generation (RAG) and GraphRAG.

- Learn about Amazon Bedrock, Amazon Neptune, and Knowledge Graph.

- Understand how AI leverages related data to improve response quality.

#### System Administrator Career Journey
- Sharing experiences from IT Helpdesk to Sysadmin.

- The roles of Linux, Networking, Cloud, and DevOps.

- Learning experiences and preparation for the job.

#### Teamwork in Projects
- Principles of effective teamwork.

- Communication, task allocation, and individual responsibilities.

- Some tools to support task management such as Trello, Slack, and Google Workspace.

#### Multiplayer on AWS
- Building a multiplayer game using WebSocket.

- Using API Gateway, Lambda, and DynamoDB to manage connections.

- Illustrating serverless architecture for real-time applications.

#### Cyber ​​Security with AWS
- Overview of AWS WAF.

- Combining Machine Learning to detect cyberattacks.

- Introducing NIDS models and monitoring systems on AWS.

#### Docker and Containerization
- Comparing Virtual Machines and Containers.

- Docker Image, Docker Container, and Dockerfile.

- Applying Docker in CI/CD and Cloud Native.

### What You Learned

#### Expertise
- Gaining a deeper understanding of AI, GraphRAG, and AI services on AWS.

- Understanding Serverless and Cloud Native architecture.

- Learning more about Docker, system security, and Machine Learning.

#### Skills

- Problem-solving thinking through real-world examples.

- Teamwork and communication skills.

- Career and learning orientation.

### Application to Work

- Practicing Docker in personal projects.

- Learning more about Amazon Bedrock and Neptune.

- Building Serverless applications on AWS.

- Applying teamwork principles to the project.

- Continuing to build my portfolio and improve my Cloud skills.

### Experience at the event

The workshop series covered a wide range of topics from AI, Cloud, DevOps to Cyber ​​Security and soft skills. The speakers not only introduced foundational knowledge but also shared many practical experiences from their studies and work. This helped me better understand how technologies are applied in businesses, rather than just focusing on theory.

What impressed me most was that each presentation included illustrative examples or real-world demos. I had the opportunity to learn about GraphRAG with Amazon Bedrock and Neptune, multiplayer architecture using AWS WebSocket, Docker applications in software deployment, and a cyberattack detection model using Machine Learning combined with AWS WAF.

Besides technical knowledge, the sharing about career development journeys and teamwork skills was also very valuable. I found that proactive learning, personal project development, and communication skills training are crucial elements for growth in the field of Information Technology.
